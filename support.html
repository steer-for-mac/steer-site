Help guide

The in-app help page covers setup, button mapping, profiles, gyro, haptics, and troubleshooting. Open Steer → Preferences → Help, or hold Command and press ? from the menu bar.

Controller not detected

Connect via USB first to confirm the controller is working, then try Bluetooth. Use “Scan for controller” in the menu bar. If Steer was recently updated, macOS may have revoked Accessibility. Check System Settings → Privacy & Security → Accessibility. Input Monitoring is a separate permission and is not needed for detection: it applies only to a DualSense Edge's rear paddles, tilt-to-aim on a DualShock 4, and off-brand pads.
