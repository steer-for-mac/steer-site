Help guide

The in-app help page covers setup, button mapping, profiles, gyro, haptics, and troubleshooting. Open Steer → Preferences → Help, or press ⌘? from the menu bar.

Controller not detected

Connect via USB first to confirm the controller is working, then try Bluetooth. Use "Scan for controller" in the menu bar. If Steer was recently updated, macOS may have revoked Input Monitoring — check System Settings → Privacy & Security.

Restoring a purchase

Open Steer → Preferences → General. If you've previously purchased, click "Restore Purchase". You must be signed in to the same Apple ID used at purchase.

Refunds

App Store purchases are refunded through Apple at reportaproblem.apple.com. See the refund page for details.
